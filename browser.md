## shortcuts

`Ctrl` `L` Goto address bar

`Ctrl` `H` History

`Ctrl` `F` Find

`Ctrl` `J` Downloads

`Ctrl` `K` Search from page

### chrome

`Ctrl` `F` Find

`Ctrl` `G` Jump to the next match

`Ctrl` `Shift` `G` Jump to  previous match

`Ctrl` `Shift` `O` Bookmark Manager

`Ctrl` `9` Goto last tab

### firefox

`Ctrl` `D` Add bookmark

`Ctrl` `I` Bookmarks
